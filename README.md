World Atlas is a responsive React web application that allows users to explore countries by browsing details such as names, regions, and flags. It includes dynamic routing, search, and filter functionalities, which enable users to search and filter countries by name and region. The app also provides an informative "About" section with interesting facts, and a "Contact Us" form for user inquiries. Built using React 19, this project integrates Axios for API calls to fetch live data, and features a custom loader and error handling for unmatched routes. With a modern and mobile-friendly UI, it uses react-router-dom v6 for routing and REST API for country data. This project follows best practices for React development and is structured with modular components for scalability. Perfect for learning React, exploring country data, and building responsive web apps with dynamic features.
